# Quiz Application

A console-based Quiz Application developed in C++ that loads questions from external text files, provides a countdown timer for each question, automatically evaluates responses, calculates scores, and displays performance feedback.

## Features

- File-based question loading
- Countdown timer for each question
- Automated scoring system
- Performance feedback based on score
- Object-oriented design
- Uses STL (`vector`) for question management
- Modular and reusable code structure

## Technologies Used

- C++
- Standard Template Library (STL)
- File Handling
- `<thread>`
- `<chrono>`
- Object-Oriented Programming (OOP)

```bash
g++ -std=c++17 main.cpp -o main
```

